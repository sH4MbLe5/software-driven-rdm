Welcome to sdRDM's documentation!
=================================



Get started with sdRDM by running the following command

::

   # Using PyPI
   python -m pip install sdRDM

Or build by source

::

   git clone https://github.com/JR-1991/software-driven-rdm
   cd software-driven-rdm
   python3 setup.py install
